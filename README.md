This project uses data extracted from the public Amazon Reviews 2023 repository. 
Due to storage limits and the massive size of the original files, the datasets were not included directly in this repository.

The specific data subsets used in this project were:
Grocery and Gourmet Food
Health and Personal Care

To run this project, access the official dataset website: [Amazon Reviews 2023](https://amazon-reviews-2023.github.io/)
Download the corresponding files, then extract and save them in this folder before running the scripts.
